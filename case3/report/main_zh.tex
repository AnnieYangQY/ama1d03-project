\section*{(a) 界定供款计划的描述及其在养老金体系中的应用}

\subsection*{1. 界定供款（DC）与界定福利（DB）}

在\textbf{界定供款（Defined Contribution, DC）}计划中，退休福利并非预先确定。雇主和/或雇员向个人账户缴付规定金额，最终福利取决于累计供款、投资回报及费用。投资和长寿风险主要由成员承担。

相比之下，\textbf{界定福利（Defined Benefit, DB）}计划按既定公式承诺福利（例如：最终薪金的一定比例乘以服务年数），大部分投资和长寿风险由计划主办者承担。

香港强积金（MPF）制度于\textbf{2000年12月1日}推出，属于强制性 DC 计划，构成香港退休保障框架的\textbf{第二支柱}：
\begin{itemize}
  \item \textbf{第一支柱：}非供款型社会保障（如长者生活津贴、高龄津贴）
  \item \textbf{第二支柱：}强积金强制及自愿供款
  \item \textbf{第三支柱：}个人储蓄、保险及家庭支持
\end{itemize}

\subsection*{2. 强积金计划类型}

根据积金局（MPFA）分类，强积金计划主要分为三类：

\begin{table}[H]
  \centering
  \caption{强积金计划类型}
  \begin{tabular}{@{}p{3.2cm}p{9.8cm}@{}}
    \toprule
    计划类型 & 说明 \\
    \midrule
    集成信托计划 & 可供参与雇主的雇员及自雇人士参加 \\
    雇主营办计划 & 仅供单一雇主的雇员参加 \\
    行业计划 & 供特定行业雇员参加（如饮食业、建造业） \\
    \bottomrule
  \end{tabular}
\end{table}

每名雇员在计划内拥有个人账户。转换工作时，累计权益可转移至新雇主的计划或保留在个人账户中。

\subsection*{3. 强制供款与自愿供款}

\textbf{强制供款（MC）：}
\begin{itemize}
  \item 雇员与雇主各供款\textbf{有关入息的 5\%}
  \item 有关入息设有下限（每月 HK\$7,100）及上限（每月 HK\$30,000）
  \item 各方最高强制供款额：每月 HK\$1,500
\end{itemize}

\textbf{自愿供款：}
\begin{itemize}
  \item 计划内的自愿供款（VC）
  \item 可扣税自愿供款（TVC）：每年最多\textbf{HK\$60,000}可在薪俸税中扣除
\end{itemize}

在 Case III 中，Peter 与 Mary 各供款 HK\$1,000（月薪 HK\$20,000 的 5\%），雇主等额配对，每月总供款为\textbf{HK\$2,000}。

\subsection*{4. 福利、提取与退休年龄}

成员一般可在\textbf{65岁}提取累计权益。其他提前提取条件包括：永久离港、完全丧失行为能力、罹患末期疾病，以及在60岁停止受雇后提前退休。

提取时，成员可选择一次性提取，或通过强积金平台将部分余额转换为年金，以获得稳定的退休收入流。

\subsection*{5. 投资选择、回报与费用}

强积金提供多种基金类型：香港、中国、亚洲及全球股票基金；混合资产；债券及保证基金；以及指数基金（如追踪恒生指数基金的 Tracker Fund）。

成员须承担\textbf{投资风险}。基金经理收取管理费（通常每年 0.5\%--2\%），长期会侵蚀净回报。Case III 中，Peter 投资于全球基金，Mary 投资于香港基金，说明在相同供款结构下，资产配置对长期结果的影响。

\subsection*{6. 税务处理与制度角色}

强制强积金供款在法定限额内可获税务豁免。TVC 为希望弥补退休收入缺口的成员提供额外税务优惠。学术及业界研究一致表明，单靠强制供款\textbf{不足以}维持退休前的生活水平，成员须以自愿储蓄及（如符合资格）政府现金福利作补充。

\subsection*{3. 优势与劣势}

\textbf{优势：}
\begin{enumerate}
  \item \textbf{雇主配对供款：}雇主 5\% 供款在不增加雇员成本的情况下使储蓄翻倍。
  \item \textbf{可携带性：}转换工作时，累计权益可随雇员转移。
  \item \textbf{投资选择：}成员可按风险偏好选择基金。
  \item \textbf{税务优惠：}TVC 每年最多 HK\$60,000 可扣税。
  \item \textbf{复利效应：}尽早开始供款者可受惠于数十年复利增长。
\end{enumerate}

\textbf{劣势：}
\begin{enumerate}
  \item \textbf{投资风险：}成员承担全部下跌风险；样本期内 Mary 的香港基金表现逊于全球基金。
  \item \textbf{替代率偏低：}即使 Peter 的强制供款替代率亦仅 35\%，低于通常建议的 60--70\%。
  \item \textbf{费用侵蚀：}管理费在 25--45 年内产生负面复利效应。
  \item \textbf{无通胀保证：}退休收入取决于不确定的市场回报。
  \item \textbf{迟供款惩罚：}Mary 11\% 的替代率表明，40岁才开始供款无法仅靠选基金弥补。
\end{enumerate}

\subsection*{5. 生活开支、通胀与额外供款}

财务规划通常以\textbf{60--70\%替代率}为目标。Peter 的强积金收入距离 60\% 目标每月尚差约 HK\$11,360；Mary 尽管退休前月薪较低，每月缺口仍约 HK\$15,269。

工资及强制供款随消费物价指数调整，但退休收入的实际购买力取决于投资回报能否跑赢通胀。45 年内 1\% 的费用差异可显著降低最终余额。

\section*{(c) 结合社会保障制度支持 DC 计划下的退休生活}

\subsection*{1. 香港三支柱框架}

与加拿大 CPP 或美国 Social Security 不同，香港\textbf{没有}全民、与收入挂钩的公共养老金。政府支持以资产审查为主：
\begin{itemize}
  \item \textbf{第一支柱：}长者生活津贴（OALA）、高龄津贴、综援
  \item \textbf{第二支柱：}强积金强制供款及 TVC
  \item \textbf{第三支柱：}私人储蓄、保险、家庭支持
\end{itemize}

\subsection*{2. 长者生活津贴（OALA）}

年满65岁的合资格香港居民可领取普通 OALA（每月 HK\$2,065）或高额 OALA（每月 HK\$4,185）（2025/26 年度），须通过收入及资产审查。单身人士高额 OALA 的资产上限约为 HK\$401,000。\textbf{尚未提取的强积金权益可能计入资产}，使余额较高的成员可能不符合资格。

\subsection*{4. 策略示例}

\textbf{Peter（2045年退休）：}
\begin{itemize}
  \item 强积金提供良好基础（替代率 35\%）。
  \item 每月增加约 HK\$1,700 TVC，可使强积金单独达到 60\% 替代率。
  \item 预期资产水平较高，可能不符合 OALA 资格，不宜依赖政府津贴。
  \item 可考虑强积金年金化以防范长寿风险。
\end{itemize}

\textbf{Mary（2025年退休）：}
\begin{itemize}
  \item 单靠强制强积金严重不足（替代率 11\%）。
  \item 即使领取高额 OALA（综合替代率 24.4\%），距离 60\% 仍有较大缺口。
  \item 应尽快开始 TVC；可考虑延迟至65岁后继续工作。
  \item 鉴于复利时间有限，须建立第三支柱储蓄。
\end{itemize}

Mary 的现实综合方案——强积金（HK\$3,437）+ 高额 OALA（HK\$4,185）+ 兼职或储蓄（HK\$5,000）——每月约 HK\$12,622，仍低于 60\% 目标（HK\$18,706）。

\section*{主要公式}

随 CPI 增长的供款累积终值：
\begin{equation}
  FV = \sum_{t=1}^{n} PMT_0 (1+g)^{t-1} \prod_{k=t+1}^{n}(1+r_k)
\end{equation}

采用 4\% 提款率的替代率：
\begin{equation}
  RR = \frac{FV \times 0.04}{Salary_{final}}
\end{equation}

每月 TVC 缺口：
\begin{equation}
  Gap_{monthly} = Target_{monthly} - \frac{FV_{MPF} \times 0.04}{12} - OALA_{monthly}
\end{equation}
