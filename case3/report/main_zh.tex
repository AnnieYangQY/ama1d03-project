\section*{(a) 界定供款计划的描述及其在养老金体系中的应用}

\subsection*{1. 界定供款与界定福利}

在界定供款计划中，供款金额事先确定，但退休福利并不确定。雇主与雇员向个人账户供款，最终余额取决于供款多少、投资表现及费用高低，成员须承担大部分投资与长寿风险。

界定福利计划则相反：福利公式固定，所需供款并不固定。主办者依精算假设厘定供款水平，计划资不抵债时须承担大部分风险。香港的职业退休计划可按 DC 或 DB 运作，与强制性强积金共同构成第二支柱。

香港强积金制度于\textbf{2000年12月1日}推出，属于强制性 DC 计划，是香港退休保障框架的\textbf{第二支柱}：
\begin{itemize}
  \item \textbf{第一支柱：}定额现金福利，须通过资产及收入审查。综援是最后安全网，长者生活津贴面向中低收入长者，高龄津贴覆盖面较广但金额较低。
  \item \textbf{第二支柱：}强积金强制及自愿供款，以及适用的职业退休计划。
  \item \textbf{第三支柱：}个人储蓄、保险及家庭支持。
\end{itemize}

香港没有与过往收入挂钩的公共养老金。第一支柱发放定额津贴，因此强积金是中产退休收入的主要基础。这一点与加拿大 CPP、美国 Social Security 等制度有本质区别。

\subsection*{2. 强积金计划类型}

积金局将强积金计划分为三类：

每名雇员在计划内拥有个人账户。转换工作时，累计权益可转移至新雇主的计划，或保留在个人账户中。

\subsection*{3. 强制供款与自愿供款}

\textbf{强制供款：}
\begin{itemize}
  \item 雇员与雇主各供款\textbf{有关入息的 5\%}。
  \item 有关入息下限为每月 HK\$7,100，上限为每月 HK\$30,000。
  \item 月薪低于 HK\$7,100 时，雇员无需供款，但雇主仍须按 HK\$7,100 的 5\% 供款。
  \item 各方最高强制供款额为每月 HK\$1,500。
\end{itemize}

\textbf{自愿供款：}
\begin{itemize}
  \item \textbf{计划内 VC} 存入现有强积金账户，不可扣减薪俸税，提取规则与强制供款相同。
  \item \textbf{可扣税自愿供款（TVC，Tax Deductible Voluntary Contributions）} 存入独立个人账户，每年最多\textbf{HK\$60,000}可在薪俸税中扣除，提取条件较灵活。
\end{itemize}

Case III 中，Peter 与 Mary 月薪均为 HK\$20,000，各供款 HK\$1,000，即工资的 5\%。雇主等额配对，每月总供款为\textbf{HK\$2,000}。

\subsection*{4. 福利、提取与退休年龄}

成员一般可在\textbf{65岁}提取累计权益。亦可因永久离港、完全丧失行为能力、罹患末期疾病，或 60 岁停止受雇后提前退休而提取。累计权益不超过 HK\$5,000 者可申请小额结余提取。成员身故时，权益付予指定受益人。

退休时可一次性提取，或通过强积金平台将部分余额转为年金。年金有助于应对长寿风险，即成员寿命超过账户余额所能支撑的时期，这也是 DC 计划的主要弱点之一。

\subsection*{5. 投资选择、回报与费用}

强积金提供香港、中国、亚洲及全球股票基金、混合资产基金、债券及保证基金，以及追踪恒生指数等指数基金。

成员须承担投资风险。基金经理一般收取每年 0.5\% 至 2\% 的管理费，长期会压低净回报。Case III 中，Peter 投资全球基金，Mary 投资香港基金，说明在相同供款结构下，基金选择仍会影响长期结果。

\subsection*{6. 税务处理与制度角色}

强制强积金供款在法定限额内可获税务豁免。TVC 为希望弥补退休收入缺口的成员提供额外税务优惠。研究与业界实践均表明，单靠强制供款\textbf{不足以}维持退休前生活水平，成员还须增加自愿储蓄，并在符合资格时申领政府现金福利。

\section*{(b) 界定供款计划的比较与分析：Peter 与 Mary 案例}

\subsection*{1. 模型设定}

本文模拟自 2000 年 12 月起至各成员年满 65 岁止的按月强积金累积过程。主要假设见表~\ref{tab:assumptions}，基金回报与 CPI 的数据来源见表~\ref{tab:sources}。

\textbf{假设说明：}
\begin{itemize}
  \item \textbf{工资随 CPI 增长。} 这相当于假设职业生涯中实际工资不变。现实中工资往往在早年增长快于通胀，会提高退休前最终工资，从而进一步压低替代率。
  \item \textbf{投资回报。} 最新数据之前采用实际月度回报，之后假设名义回报率每年\textbf{4\%}，与积金局及业界常用的长期规划假设一致。本模型并非纯粹历史回测。2000 至 2025 年全球基金跑赢港股，只反映某一历史阶段，不应视为未来预测。
  \item \textbf{4\% 提款法则。} 退休第一年的收入等于账户余额的 4\%，并假设此后提款随通胀上升。该法则最初针对美国 30 年退休期设计。香港居民预期寿命常超过 85 岁，固定提取率可能令人面临资金耗尽的风险。
  \item \textbf{费率。} 本文假设每年 1\%，处于强积金常见区间的中位。指数基金费率常低于 0.5\%，主动股票基金多在 1.2\% 至 1.8\% 之间。供款 25 至 45 年期间，费率差异会逐步放大。
\end{itemize}

\subsection*{3. 优势与劣势}

\textbf{优势：}
\begin{enumerate}
  \item \textbf{个人账户、可携带。} 权益归成员所有并随其转移，适合香港流动性高的劳动力市场，亦无 DB 计划中的雇主信用风险。
  \item \textbf{雇主配对供款。} 雇主 5\% 供款在不增加雇员成本的情况下使储蓄翻倍。
  \item \textbf{投资选择。} 成员可按风险偏好选择基金。
  \item \textbf{TVC 税务优惠。} 每年最多 HK\$60,000 TVC 可扣税。
  \item \textbf{长期复利。} 尽早供款受益最大。按年化 4\% 计算，72 法则意味着财富约每 18 年翻一番。45 年累计约为本金的 6 倍，25 年则约为 2.7 倍。
\end{enumerate}

\textbf{劣势：}
\begin{enumerate}
  \item \textbf{市场与序列风险。} 成员承担全部下跌风险，退休前市场急跌会大幅削减最终余额。
  \item \textbf{无风险共担。} 与 DB 计划不同，DC 计划不在成员之间分摊长寿或投资不足的风险。
  \item \textbf{替代率偏低。} 即使 Peter 仅靠强制供款也只有 36\% 替代率，低于财务规划常用的 60\% 至 70\% 区间。
  \item \textbf{费用与不当操作。} 管理费长期侵蚀回报，频繁换基金往往进一步降低净收益。
  \item \textbf{迟供款代价高。} Mary 9.1\% 的替代率表明，40 岁才开始供款无法单靠挑选更好的基金来弥补。
\end{enumerate}

\section*{(c) 结合社会保障制度支持 DC 计划下的退休生活}

\subsection*{1. 香港三支柱框架}

与加拿大 CPP 或美国 Social Security 不同，香港没有与收入挂钩的全民养老金。政府支持须通过资产审查，并呈分层结构：
\begin{itemize}
  \item \textbf{第一支柱}为低资产、低收入长者提供基本保障。综援审查最严，长者生活津贴受资产及收入限制，高龄津贴较易获批但金额较低。
  \item \textbf{第二支柱}是就业人士的强制性强积金。
  \item \textbf{第三支柱}是大多数人在退休后维持生活水平的主要依靠。
\end{itemize}

对 Peter、Mary 这类中产雇员，强积金余额一旦计入资产审查，第一支柱福利大多无法申领。退休收入须来自强积金与自愿储蓄，不能指望把各支柱简单相加。

\subsection*{2. 长者生活津贴}

年满 65 岁的合资格香港居民，在通过收入及资产审查后，可于 2025/26 年度领取普通 OALA 每月 HK\$2,065 或高额 OALA 每月 HK\$4,185。单身人士两类津贴的资产上限约为 HK\$401,000。\textbf{尚未提取的强积金权益计入资产。} DC 计划积累越多，越难获得 OALA，两种支持并不能直接叠加。

\subsection*{3. 综合退休收入}

\subsection*{4. 策略示例}

\textbf{Peter，2045 年退休：}
\begin{itemize}
  \item 强积金已提供 36\% 的合理基础，但按预期强积金资产将\textbf{不符合 OALA 资格}。
  \item 每月通过\textbf{可扣税自愿供款（TVC）}增加约 HK\$1,600（TVC 定义见 (a) 第 3 节），可在每年 HK\$60,000 扣税上限内使强积金收入达到 60\% 替代率。
  \item 全程持有全球股票并非最合理做法。年轻时可提高权益比重，55 岁后逐步转向债券及保本基金，有助于降低序列风险。
  \item 将部分余额转为年金，可形成保底终身收入，应对长寿风险。
\end{itemize}

\textbf{Mary，2025 年退休：}
\begin{itemize}
  \item 单靠强制强积金仅 9.1\%。余额 HK\$851,000 高于 OALA 资产上限 HK\$401,000，\textbf{同样无法领取 OALA}。
  \item 延迟至 70 岁退休，模拟中替代率仅升至 10.9\%。多供几年有帮助，但无法弥补比 Peter 晚开始 20 年的损失。
  \item 所需额外储蓄远超 TVC 扣税上限，须结合不可扣税 VC、第三支柱储蓄及可能的兼职收入。
  \item 账户规模较小时，部分年金化或延迟提取仍可改善收入保障。
\end{itemize}

Mary 较可行的安排是强积金月收入 HK\$2,836，加上第三支柱储蓄或兼职收入约每月 HK\$5,000，合计约 HK\$7,836，仍低于 60\% 目标 HK\$18,706。即便表~\ref{tab:income_hyp} 中假设性高额 OALA 情景，替代率亦仅 22.5\%。

香港的 DC 制度将大部分退休责任放在个人身上。Peter 的案例说明，尽早并持续供款是最有效的手段。OALA 帮助低资产长者，但对有可观强积金积累的中产存在挤出效应。\textbf{Peter 与 Mary 均须增加额外供款，Mary 缺口更大、时间更紧。}

\subsection*{6. 核心结论}

\begin{enumerate}
  \item 单靠强制强积金无法维持退休前生活水平，自愿储蓄必不可少。
  \item 在 DC 计划中，\textbf{供款起始时间}比基金选择更重要，因为时间对复利的影响大于回报率差异。
  \item 香港第一支柱是低资产长者的安全网，不能为有可观强积金余额的中产提供补充。
  \item 完整的退休规划应包括强制强积金、限额内 TVC、第三支柱储蓄、临近退休时逐步降风险，以及部分年金化。
\end{enumerate}

\section*{主要公式}

随 CPI 增长的供款累积终值，按月模拟：
\begin{equation}
  FV = \sum_{t=1}^{n} PMT_0 (1+g)^{t-1} \prod_{k=t+1}^{n}(1+r_k)
\end{equation}
其中 $PMT_0$ 为初始月度供款，$g$ 为供款年增长率，$r_k$ 为第 $k$ 期净回报率，$n$ 为供款总期数。

当 $r$ 与 $g$ 为常数且 $r \neq g$ 时，增长年金闭式解为：
\begin{equation}
  FV = PMT_0 (1+r) \cdot \frac{(1+r)^n - (1+g)^n}{r - g}
\end{equation}

4\% 提款率下的替代率：
\begin{equation}
  RR = \frac{FV \times 0.04}{Salary_{final}}
\end{equation}

要达到目标替代率 $\rho^*$，对 $PMT_{TVC}$ 求解：
\begin{equation}
  \rho^* = \frac{0.04 \times FV(PMT_0, g, r, n) + 0.04 \times FV(PMT_{TVC}, g, r, n')}{Salary_{final}}
\end{equation}
其中 $n'$ 为额外供款期数，本案例中等于完整供款年期。表~\ref{tab:tvc} 数值由模拟模型二分搜索求得。

达到 60\% 目标的月度收入缺口：
\begin{equation}
  Gap_{monthly} = 0.60 \times \frac{Salary_{final}}{12} - \frac{FV_{MPF} \times 0.04}{12}
\end{equation}
