\section*{(a) DC 计划与香港强积金制度}

\subsection*{1. DC 与 DB}

在\textbf{界定供款（DC）}计划中，供款确定、退休福利不确定，终值取决于供款、投资回报及费用，成员承担大部分投资与长寿风险。在\textbf{界定福利（DB）}计划中，福利公式固定而供款不确定，主办者承担大部分资不抵债风险。香港强积金于\textbf{2000年12月1日}推出，为强制性 DC 计划，与适用的职业退休计划共同构成\textbf{第二支柱}。

香港没有与收入挂钩的公共养老金。\textbf{第一支柱}为定额、经审查的现金福利：综援为最后安全网，长者生活津贴（OALA）面向中低收入长者，高龄津贴（OAA）覆盖面较广但金额较低。\textbf{第三支柱}为个人储蓄、保险及家庭支持。对中产而言，强积金是退休收入的主要基础。

\subsection*{2. Case III 相关的强积金规则}

积金局将计划分为集成信托、雇主营办及行业计划，成员拥有可携带的个人账户。

\textbf{强制供款：}雇员与雇主各供\textbf{5\%}有关入息，下限 HK\$7,100、上限 HK\$30,000。月薪低于 HK\$7,100 时雇员免供，雇主仍按 HK\$7,100 的 5\% 供款。各方最高 HK\$1,500/月。

\textbf{自愿供款：}计划内 \textbf{VC} 无薪俸税扣除，提取规则同强制供款。\textbf{可扣税自愿供款（TVC）}为独立账户，每年最多\textbf{HK\$60,000}可扣税。

成员一般\textbf{65岁}可提取，亦可因离港、丧失行为能力或末期疾病提前提取，或将部分余额转为年金以应对长寿风险。基金年费通常为 0.5\%--2\%。Case III 中 Peter 与 Mary 各供 HK\$1,000、雇主配对，每月合计\textbf{HK\$2,000}。Peter 投资全球股票基金，Mary 投资香港股票基金。单靠强制供款\textbf{不足以}维持退休前生活水平。

\subsection*{3. 优势与劣势}

\textbf{优势：}
\begin{enumerate}
  \item 个人账户、可携带，适合香港高流动性劳动力市场。
  \item 雇主配对在不增加雇员成本的情况下使储蓄翻倍。
  \item 成员可按风险偏好选择基金。
  \item TVC 每年最多 HK\$60,000 可扣税。
  \item 早供款复利显著；年化 4\% 时财富约每 18 年翻一番。
\end{enumerate}

\textbf{劣势：}
\begin{enumerate}
  \item 成员承担全部市场与序列风险。
  \item 与 DB 不同，无长寿或投资短缺的风险共担。
  \item 即使 Peter 仅靠强制供款亦仅 36\% 替代率。
  \item 费用在 25--45 年内持续侵蚀回报。
  \item Mary 9.1\% 表明 40 岁才开始供款无法单靠选基金弥补。
\end{enumerate}

\section*{(c) 社会保障与建议}

\subsection*{1. 三支柱与 OALA}

与加拿大 CPP 或美国 Social Security 不同，香港没有全民、与收入挂钩的养老金。对 Peter、Mary 这类中产，强积金余额计入资产审查后，第一支柱福利大多无法申领。

OALA 2025/26 普通津贴每月 HK\$2,065、高额津贴 HK\$4,185，须通过收入及资产审查。单身上限约 HK\$401,000。\textbf{未提取强积金计入资产。}DC 积累越多，越难获得 OALA。

\subsection*{3. 策略与结论}

\textbf{Peter，2045 年退休：}
\begin{itemize}
  \item 强积金提供 36\% 基础，但无法领取 OALA。
  \item 每月通过 TVC 增加约 HK\$1,600 可达 60\%，在扣税上限内。
  \item 55 岁后逐步转向债券及保本基金，降低序列风险。
  \item 考虑部分年金化以防范长寿风险。
\end{itemize}

\textbf{Mary，2025 年退休：}
\begin{itemize}
  \item 强制强积金仅 9.1\%；余额 HK\$851,000，无法领取 OALA。
  \item 延至 70 岁退休替代率仅升至 10.9\%（表~\ref{tab:delay}）。
  \item 须靠第三支柱储蓄及兼职；单靠 TVC 不现实。
\end{itemize}

\textbf{核心结论：}
\begin{enumerate}
  \item 单靠强制强积金无法维持退休前生活水平。
  \item \textbf{供款起始时间}比基金选择更重要。
  \item 第一支柱是低资产长者安全网，不能补充中产。
  \item 完整规划须结合强积金、限额内 TVC、第三支柱储蓄、生命周期降风险及部分年金化。
\end{enumerate}

\section*{主要公式}

随 CPI 增长的供款累积终值：
\begin{equation}
  FV = \sum_{t=1}^{n} PMT_0 (1+g)^{t-1} \prod_{k=t+1}^{n}(1+r_k)
\end{equation}
当 $r$、$g$ 为常数且 $r \neq g$ 时，
\begin{equation}
  FV = PMT_0 (1+r) \cdot \frac{(1+r)^n - (1+g)^n}{r - g}
\end{equation}
替代率及距 60\% 目标的月缺口：
\begin{equation}
  RR = \frac{FV \times 0.04}{Salary_{final}}, \qquad
  Gap_{monthly} = 0.60 \times \frac{Salary_{final}}{12} - \frac{FV \times 0.04}{12}
\end{equation}
其中 $PMT_0$ 为初始月供款，$g$ 为供款增长率，$r_k$ 为第 $k$ 期净回报，$n$ 为期数。
